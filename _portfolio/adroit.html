The client's requirements for the project were to develop a concept for an online store with simple design and layout. The application was built using Ruby on Rails with the front end styled using HTML, CSS animations, Jquery, and Bootstrap. A custom cart feature was added to allow customers to add or remove items available for purchase. Payment integration was done using Stripe enabling customers to purchase items with ease.
